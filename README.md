# Summary
A boilerplate for MERN stack!

# Technologies Used
* Mongodb
* Express
* React

# Backend Routes
METHOD | URL | Purpose
--- | --- | ---
POST | /auth/signup | Adds new user to user database
POST | /auth/login | Authenticates login details
POST | /auth/me/from/token | Checks if token is present on browser refresh
